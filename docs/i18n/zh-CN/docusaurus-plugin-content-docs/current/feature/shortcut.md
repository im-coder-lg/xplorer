# 快捷键

| 关键                  | 快捷方式         |
| ------------------- | ------------ |
| `Ctrl + W`          | 退出 Xplorer   |
| `Ctrl + E`          | 关闭标签页        |
| `Ctrl + C`          | 复制           |
| `Ctrl + V`          | 粘贴           |
| `Ctrl + X`          | 剪切           |
| `Ctrl + A`          | 选择全部         |
| `Ctrl + H`          | 切换隐藏文件       |
| `Ctrl + O`          | 预览文件         |
| `Ctrl + R`          | 更新           |
| `Ctrl + P`          | 文件属性         |
| `Ctrl + F`          | 查找文件/文件夹     |
| `Ctrl + T`          | 新选项卡         |
| `Alt + O`           | 使用…打开        |
| `Alt + N`           | 新文件          |
| `Alt + S`           | 创建快捷方式       |
| `Alt + Left Arrow`  | 上一页          |
| `Alt + Right Arrow` | 下一页          |
| `Alt + Up Arrow`    | 最大化          |
| `Alt + Down Arrow`  | 最小化          |
| `Alt + T`           | 在终端中打开       |
| `Alt + P`           | 添加到面板        |
| `Alt + Shift + C`   | 复制文件位置       |
| `F2`                | 改名           |
| `F5`                | 更新           |
| `F10`               | 最小化          |
| `F11`               | 最大化          |
| `Shift + N`         | 新文件夹         |
| `Shift + Del`       | 永久删除         |
| `Shift + Enter`     | 在 VSCode 中打开 |
| `Enter`             | 打开           |
| `Del`               | 删除文件         |
