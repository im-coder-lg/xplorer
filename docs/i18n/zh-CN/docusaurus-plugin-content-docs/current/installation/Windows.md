## 面对Windows Defender吗？
这实际上不是一个错误，它是微软的设计选择，以便保护我们中那些陌生科技的人。 你不需要担心Xplorer在这种情况下的安全。 因为它是 [开源的](https://github.com/kimlimjustin/xplorer) ，您可以检查代码，甚至可以建立自己的版本！

要处理这个问题，你只需点击 `更多信息` 按钮，然后点击任意运行。
