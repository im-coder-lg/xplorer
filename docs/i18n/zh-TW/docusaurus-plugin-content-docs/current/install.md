<summary>
How to install in Arch OS?
</summary>

Run following command:

```bash
sudo pacman -u [安裝檔案名稱]
```

:::info If you faced the `xplorer exists in filesystem` error, run this command instead:

```bash
sudo pacman -u [安裝檔案名稱] --overwrite "*"
```

:::
